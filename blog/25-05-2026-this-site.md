# How I Built This Website

*May 25, 2026*

## Thought Process

I was trying to look for a website renderer that worked around Markdown files. Github pages exists, however I wasn't a fan of being forced to deploy onto Github (considering their reliability lately).

* My original thought was to use [BearBlog](https://bearblog.dev/), because it was simple and didn't require self-hosting. However, it was limited in what you could do, and it required you to pay in order to upload media (which I think is ridiculous)
* My next thought was to browse [awesome-selfhosted](https://github.com/awesome-selfhosted/awesome-selfhosted#blogging-platforms) because 295,000 stars can't be wrong? But it didn't really suit exactly what I wanted
* I ended up settling with [compiiile](https://github.com/compiiile/compiiile), which is a (relatively) small markdown website renderer. I really enjoy it's simplicity but support for markdown, folders/hierarchy, PDFs and Slideshows.

## Deploying

My website is still uploaded to my [github pages repository](https://github.com/maxcraig112/maxcraig112.github.io/tree/main), however I am publishing it using Cloudflare Pages, as it is free and more reliable than Github.

I purchased the domain from GoDaddy.
